# Aditya Singh Rajput's Portfolio

A 3D interactive portfolio built with Three.js and GLSL shaders, featuring a room with monitors, an arcade machine, a whiteboard, and a Rubik's cube. The left monitor shows an About Me OS simulation, the right monitor shows a 3D art gallery of projects, and the arcade machine has playable games.

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
