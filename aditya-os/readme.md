# Aditya's OS

A simulated Windows OS experience — the About Me section of the portfolio. Features an Instagram-style story feed, experience timeline, contact form, and resume download.

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
