# Aditya's Art Gallery

A virtual first-person 3D art gallery showcasing all projects as museum exhibits. Walk around and interact with each painting to view project details, source code, and live demos.

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
