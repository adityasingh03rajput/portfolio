# Aditya's Arcade Machine

Playable arcade games built in vanilla JS — Snake, Tetris, and Breakout. Embedded inside the 3D portfolio's arcade machine.

### Portfolio
- [Source Code](https://github.com/adityasingh03rajput/portfolio)

## Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
